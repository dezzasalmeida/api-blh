Informações sobre o Projeto
----------------------------

Esta API faz parte de um projeto educacional para um projeto de conclusão de curso de pós-graduação em Desenvolvimento Full Stack da aluna Andrezza Almeida. O projeto visa promover o aleitamento materno e a doação de leite humano. O projeto não é comercial.
